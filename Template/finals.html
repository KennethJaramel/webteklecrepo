PHP SuperGlobals:

$GLOBALS- used to access global variables from anywhere in the PHP script

$_FILES- for uploading multiple files.

$_ENV- is used to return the environment variables from the web server.

$_COOKIE- Cookies are small text files loaded from a server to a client computer storing some information regarding the client computer.

$_SESSION- are wonderful ways to pass variables.
